## findMP-a visual desktop application that detect malicious npm package

#### develop

1. install python3 (>= 3.9)

2. install python skearn (>=1.2.1)

3. install node (>=16)

4. install npm dependency. 

```bash
npm install
```

4. start app 

```bash
npm run start
```

#### generate platform-specific application

```bash
npm run make
```
